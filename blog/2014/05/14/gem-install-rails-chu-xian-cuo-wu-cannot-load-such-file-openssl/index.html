Gem Install 出现错误 Cannot Load Such File -- Openssl

今天想要在cocoapods加入一个库，用pod install后出现这个错误，百思不得其解，上网找了一圈发现，这个问题应该是更新了Mac OS X导致的，解决方法如下：

brew install openssl rvm pkg install openssl rvm reinstall [version]

以上命令执行一遍，问题解决。
